Dreiecksrechner

Das Programm entstand als eine Einführung in Trigonometrie und TKinter. Da die meisten Programme, welche zur Berechnung von trigonometrischen Formeln im Internet zu finden sind, entschied ich mich dazu ein eigenes Programm zu schreiben, um Python und TKinter zu lernen.


Sobald man das Programm startet, kann man eine von drei Formeln auswählen. Danach kann man in den Textfeldern darunter die Seitenlängen und Winkel des Dreieckes angeben. Wenn der Knopf gedrückt wird, erscheint eine Zeichnung des angegeben Dreiecks. Es wird ebenfalls die gesuchte Variable ausgegeben.

Farbkonverter

Das Programm entstand als eine Einführung in die Tiefen der Farbtheorie und die Verschiedene Darstellungsmöglichkeiten einer Farbe. Es hat mir geholfen zu verstehen, wie Farben in der Informatik dargestellt werden und wie man von einer Darstellungsmöglichkeit zur anderen kommt.


Nachdem Start des Programms kann man eine Farbsystem wählen von dem aus eine Farbe in zwei andere Farbsysteme konvertiert wird. Teil der Farbsysteme sind HSV, RGB und HEX. Danach kann man eine Farbe eingeben und sie mit Hilfe des Programms konvertieren. Im unteren Teil des Fensters wird die Farbe angezeigt.

Projekt-R

In diesem Projekt verwenden wir einen Raspberry Pi und Python, um eine Roboterarm zu steuern. Der Roboterarm verfügt über 3-Freiheitsgrade und einen Finray-Greifmechanismus. Ein Bildanalyse Programm verarbeitet die eingaben einer Kamera, um die Position des zu greifenden Objekts zu erkennen. Um den Roboter zu bewegen, berechnen wir mit Hilfe von Inverser Kinematik, die Motorpositionen.

Projekt-LoL-Randomizer

Projekt-L ist eine Webseite welche dazu dient einen zufälligen Champion und Build für das Spiel League of Legends auszuwählen. Es verwendet eine große JSON-Datei welche alle Daten speichert welche für die 162 Champions und 105 Items sowie 16 Runen benötigt werden. Ein JavaScript liest die Daten aus und ein kleiner Alogrithmus stell diese in der richtigen Reihenfolge auf um sie dann im UI der Webseite anzuzeigen.
